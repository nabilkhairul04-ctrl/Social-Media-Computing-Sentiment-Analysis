## Project Description

This project develops an end-to-end Natural Language Processing (NLP) pipeline for customer review analysis using the Flipkart Product Reviews Dataset.

The project includes:

- Sentiment Analysis
- Opinion Mining
- Aspect-Based Sentiment Analysis (ABSA)
- Machine Learning Models
- Deep Learning Models
- Transformer Models

## Models Implemented

### Lexicon-Based

- VADER

### Machine Learning

- Naïve Bayes
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

### Deep Learning

- BiLSTM
- GRU

### Transformer

- DistilBERT
- RoBERTa
